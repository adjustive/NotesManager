Происходит создание кнопки и установка ей свойств, переданных в параметрах.

QWidget * widget = new QWidget();
widget->show();

Функция создает кнопку и возвращает указатель на нее.

Происходит создание кнопки и установка ей свойств, переданных в параметрах. Вот так будет создана кнопка:

QToolButton * button = new QToolButton( 0 );
button->setText( "Нажми на меня" );
button->setToolTip( "Подсказка" );
button->setStatusTip( "Состояние" );
button->setWhatsThis( "Кнопка" );
button->setIcon( QIcon( ":/icon" ) );
button->setCheckable( true );

Тоже самое, но с использование данной функции:

QToolButton * button = Create::Button::get( 0, "Нажми на меня", "Подсказка", "Состояние", "Кнопка", QIcon( ":/icon" ), true );
Аргументы
parentуказатель на виджет-родитель
textтекст на кнопке
toolTipтекст всплывающей подсказки
statusTipсостояние виджета
whatsThisтекст подсказки What's This виджета
iconиконка на кнопке
checkableданное свойство указывает, является-ли кнопка контрольной
Возвращает
указатель на созданную в функции кнопку
